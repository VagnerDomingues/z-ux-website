Roadmap

  • u.env

    safe store for credentials

    Nov 2025

    Vault for API keys, secrets and environment variables inside Unity—team sharing without ever committing raw values. Injects at build / runtime through secure bridges so nothing sensitive ships in game binaries, mitigating reverse‑engineering & accidental leak risk.

  • uDeploy

    one click to ship

    Nov 2025

    Single‑click deployment hub: builds, versions, bundles & uploads Unity targets (WebGL / Desktop / Mobile) to FTP / SFTP / SSH endpoints with checksum validation and auto‑generated release notes. Roadmap: cloud drivers (AWS / GCP / Azure) + phased & scheduled rollouts.

  • uDeploy (PATCH pipeline)

    one click to patch & ship for PATCH launcher

    Nov 2025

    Post‑build chain integrating the third‑party PATCH Launcher: generates binary diffs, computes optimal patch routes, uploads manifests + payloads (game + self‑updating launcher) over CDN/FTP/SFTP, and enables atomic rollback. Includes integrity hashing & telemetry hook points.

  • uGit

    Track git changes inside unity editor

    Fev 2026

    Inline Git control: per‑asset & scene hierarchy diffs + serialized Inspector property change tracking using git diff parsing; stage / commit / branch / revert without context switching. Planned: one‑click Inspector property undo + time‑travel previews to avoid “lost scene” moments.

  • uDungeon

    S Tier Procedural Generator

    Mar 2026

    Runtime map renderer for shunDungeon: consumes exported JSON + seed to expand a multidimensional byte matrix into instanced prefabs (terrain, portals, elevators, puzzles, loot) across up to 256 entity archetypes. Supports merging authored rooms, directional spawn scans & infinite reproducible variants.

  • uAlly

    LLM integrated with MCP inside unity

    Mar 2026

    MCP‑backed LLM co‑pilot: context‑aware chat (open scene, selected objects, components), script & editor tool scaffolding, asset naming, log triage and refactor suggestions. Extensible function registry for custom runtime commands; provider‑agnostic layer + offline cache on roadmap.

  • uBets

    Blockchain bet api inside unity

    Jun 2027

    On‑chain wagering SDK: create, sign, submit & settle bets from gameplay with verifiable odds math and escrow / result validation. Markets auto‑listed to partner platforms (e.g. bitcoin-betting.com) for liquidity. Wallet abstraction, oracle feeds & audit trail export supported / expanding.
