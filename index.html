Roadmap

  • u.env

    safe store for credentials

    Nov 2025

    Safe vault for API keys, secrets and environment variables inside Unity—team sharing without ever committing raw values. The .env is excluded from build & runtime it only runs on editor so nothing sensitive ships in game binaries, mitigating reverse‑engineering & accidental leak risk.

  • uDeploy

    one click to ship

    Nov 2025

    Single‑click deployment build & uploads to server distribution using FTP or SCP (over SSH).

  • uDeploy (PATCH)

    one click to patch & ship

    Nov 2025

    Post‑build chain integrating the third‑party PATCH launcher: generates binary diffs routes, computes optimal patch routes, uploads manifests + payloads. Easy to setup pipeline for game or launcher self updates. Uploads to server using FTP or SCP (over ssh). Enables atomic rollback by cleaning up server storage and preparing launcher to restore the game. And the most wanted feature a per/dev launcher compatibility aimed for playtest driven game-development.

  • uGit

    Track git changes inside unity editor

    Fev 2026

    Inline Git control: per‑asset & scene hierarchy diffs + serialized Inspector property change tracking using git diff parsing; stage / commit / branch / revert without context switching. one‑click Rollback in any Inspector property undo + time‑travel previews. never feel lost in minor tweaks again.

  • uDungeon

    An S Tier Procedural Generator

    Mar 2026

    Runtime map renderer based on the shunDungeon generator: consumes exported JSON + seed to expand a multidimensional byte matrix into instanced prefabs (terrain, portals, elevators, puzzles, loot) across up to 256 entity archetypes. Supports merging authored pre-designed rooms for robust puzzles, directional spawn scans & infinite reproducible variants.

  • uAlly

    LLM partner inside your game engine

    Mar 2026

    MCP‑backed LLM co‑pilot: context‑aware chat (open scene, selected objects, components), script & editor tool scaffolding, asset naming, log triage and refactor suggestions. Extensible function registry for custom runtime commands; compatible with major models out there , chatgpt, claude, grok & gemini.

  • uBets

    Easy to use Blockchain bet api

    Jun 2027

    On‑chain wagering SDK: create, sign, submit & settle bets from gameplay with auto odds generation based on your player winnings/losses math. Markets auto‑listed to provider platforms (e.g. bitcoin-betting.com) for improved liquidity. Wallet abstraction and easy in-game integration
